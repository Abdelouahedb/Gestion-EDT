# Gestion EDT

Application web de gestion des emplois du temps scolaires.

🔗 **Démo en ligne :**  
http://gestion-edt.page.gd

## Tech Stack

`HTML` `CSS` `JavaScript` `PHP` `MySQL` `PDO`

## Fonctionnalités

- Authentification avec rôles
- Dashboard administrateur
- Gestion des filières, semestres, enseignants, salles, modules et créneaux
- Création des emplois du temps
- Détection des conflits
- Planning étudiant
- Planning enseignant
- Impression / export PDF

## Installation locale

```txt
1. Copier le projet dans htdocs
2. Lancer Apache et MySQL
3. Créer la base gestion_emploi_mgsi
4. Importer projet.sql
5. Ouvrir http://localhost/EMPLOIS_MANAGE/
```

## Sécurité

- Mots de passe hashés
- Connexion avec sessions PHP
- Requêtes préparées avec PDO
